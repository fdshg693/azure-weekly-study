# Azure 学習レポジトリ

- `commands`
    - 便利コマンドのメモ

- `learn/{topic}`
    - Azureの各トピックに関する学習をまとめるフォルダ。
    - トピックごとにプロジェクトが分かれ、各プロジェクトでコードを通して学習を進める形式をとる。
        - 徐々にプロジェクトを難しくしていき、徐々にトピックについての理解を深めていく形式をとる。

- `local/{topic}`
    - ローカルで動かすプロジェクトをまとめるフォルダ。
    - Azureのリソースをコードで構築するのではなく、ローカルで完結するプロジェクトはこちらに置く。

- `projects/{project-name}`
    - 具体的な学習プロジェクトをまとめるフォルダ。
    - プロジェクトごとに、Azureのリソースをコードで構築し、学習を進める。
    - サービス単位を超えた学習・あるいは実際に作りたいものを作るプロジェクトなど、トピック配下に置きづらいプロジェクトはこちらに置く。
