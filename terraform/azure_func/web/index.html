Azure Functions x HTMX 乱数デモ

Azure Static Web Apps から配信したページから、HTMX で Azure Functions を呼び出す。 同期版と、Logic App + Service Bus + Worker Function の非同期版の 2 系統。

同期版 — /api/random

Function App が直接乱数を返す。即時応答。

—

非同期版 — /api/async-random プロキシ

ブラウザ → Function プロキシ → Logic App → Service Bus → Worker（数秒スリープ）→ Table → 結果。
Worker のスリープぶん（既定 5 秒）＋ Logic App の Until ポーリングで体感数秒待つ。
