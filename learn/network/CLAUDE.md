# ネットワークについて学習する

- `./basic`
    - 実際にAzureでネットワークに関する構築を行い、通信一般＋Azureにおける通信/構成を学習する（実行はユーザーが行い、AIが実行することはないことに注意）
    - 言語はBicep, just, azure CLIを利用
        - just により、コマンドの簡略化・まとめ
        - 環境構築等は可能な限り、Bicepで、Bicepがカバーできないあるいはすべきでない箇所はazure cliを利用

- `./advanced`
    - ネットワークに関するより高度な内容を学習するためのコードや資料

- `./memo`
    - ローカルで様々なコマンドを叩いてネットワーク通信について学習用のメモ
    - Git 管理から除外する
    - ユーザーからの指示がなければ触らない
